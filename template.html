2. Create

To create a page, select “Page” from the “Create” drop down menu

  1. Enter a page title
  2. Add an Image or Slideshow to the top of the page (optional) (If both Large Image and Slideshow are checked, only the Multiple Image slideshow will display)


    Large Image
    1. Check “Add Large Image” on the page (Figure 1)
    2. Select the image using the browse option add the Alternative and Title text, and crop
      1. Add an Alt tag to the Alternate text field
      2. Add a Title tag to the Title field
      3. Click “Manage image crops”
      4. Select the crop area to display
      5. Click “Apply Crop” on the pop up
    3. To make into a slideshow, click on the “Add More” values button
    4. Repeat until the desired numbers of items are created
