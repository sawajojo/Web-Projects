2. Create

Pages

Page Features

To create a page, select “Page” from the “Create” drop down menu

  1. Enter a page title
  2. Add an Image or Slideshow to the top of the page (optional) (If both Large Image and Slideshow are checked, only the Multiple Image slideshow will display)


    Large Image
    1. Check “Add Large Image” on the page (Figure 1)
    2. Select the image using the browse option add the Alternative and Title text, and crop
      1. Add an Alt tag to the Alternate text field
      2. Add a Title tag to the Title field
      3. Click “Manage image crops”
      4. Select the crop area to display
      5. Click “Apply Crop” on the pop up
    3. To make into a slideshow, click on the “Add More” values button
    4. Repeat until the desired numbers of items are created
    OR

    Multiple Image Slideshow


    1. Check the “Add slideshow” box (Figure 4)
    2. Enter a title for the slideshow image, in the Title box
    3. Select the image using the browse option, add the Alternative and Title text, and crop
      1. Add an Alt tag to the Alternate text field
      2. Add a Title tag to the Title field
      3. Click “Manage image crops”
      4. Select the crop area to display
      5. Click “Apply Crop” on the pop up
    4. To link this image, enter the URL in the Link box (e.g. http://www.buffalostate.edu)
    5. Click on the “Add More” values button
    6. Repeat until the desired numbers of items are created.
  3. 1 column page (Figure 1)
    1. Enter the content into the Content Area box
  4. Adding 2 or 3 Columns to the page (optional) (Figure 2)
    1. Select the Add Columns check box
    2. Select the image using the browse option, add the Alternative and Title text, and crop
      1. Add an Alt tag to the Alternate text field
      2. Add a Title tag to the Title field
      3. Click “Manage image crops”
      4. Select the crop area to display
      5. Click “Apply Crop” on the pop up
    3. Enter a title for the column
    4. Enter the column content
    5. Repeat for Column 2 and Column 3 for a 3 column page. For a 2 column page leave Column 3 blank.
  5. Adding the List feature to a page (optional) (Figure 3)
    1. Select “Add list items”
    2. Select the image using the browse option, add the Alternative and Title text, and crop
      1. Add an Alt tag to the Alternate text field
      2. Add a Title tag to the Title field
      3. Click “Manage image crops”
      4. Select the crop area to display
      5. Click “Apply Crop” on the pop up
    3. Enter the title for the list item
    4. Enter the list text
    5. To link this list item, enter the URL in the External Link box (e.g. http://www.buffalostate.edu)
    6. To add another list item, click the “Add more values” button
    7. Repeat until the desired numbers of items are created.
  6. Under “Menu,” select where the page should appear
    Select “Place Below” when hovering to position the page.

  7. To add sub navigation, click the “Place as child”. To add to existing sub navigation, click the left arrow to expand the menu, and select “Place below” to position.

    Hit “Save”

Faculty and Staff


Select Faculty from the Create drop down menu

  1. To select an existing faculty member
    1. Select the correct name from the faculty box
    2. Select the image using the browse option, add the Alternative and Title text, and crop
      1. Add an Alt tag to the Alternate text field
      2. Add a Title tag to the Title field
      3. Click “Manage image crops”
      4. Select the crop area to display
      5. Click “Apply Crop” on the pop up
    3. Enter the faculty member’s content in the Information box
    4. Click Save
  2. To create a new faculty member
    1. Select the Add faculty member box
    2. Select the image using the browse option, add the Alternative and Title text, and crop
      1. Add an Alt tag to the Alternate text field
      2. Add a Title tag to the Title field
      3. Click “Manage image crops”
      4. Select the crop area to display
      5. Click “Apply Crop” on the pop up
    3. Enter the First Name
    4. Enter the Middle Initial
    5. Enter the Last Name
    6. Enter the Title
    7. Enter the Department
    8. Enter the Phone Number
    9. Enter the Office
    10. Enter the E-mail
    11. Click Save

Creating an External Link in the Menu


To create an external link to add to your menu, select “Page (Link Only) ” from the “Create” drop down menu

  1. Enter the URL under “Path”
  2. Enter the Text to display under the “Menu Link Title”
  3. If the link is a main navigation item, leave the setting at “Primary Links”
    1. If the external link is a sub-navigation item, under “Parent item” drop down, select the Parent page
  4. Hit “Save”
  5. You can reorder the navigation items by selecting the “manage” tab and following the instructions in the Manage section.

3. Manage/Edit

The elements of the site can be edited and modified using the “Manage” drop down menu

  • The “Manage” heading allows you to reorganize your site structure and navigation using drag-and-drop functionality.
    Select “Save configuration” when finished.
  • Editing: When logged in, any page can be edited while viewing. Simply click the edit tab to change your content.
    Pages can also be edited from the Manage > Pages tab

4. Files


Use the files tab to upload and organize your images, documents, and video.

Adding files

**Files cannot have spaces or special characters in their file name or else they cannot be used on the website. Once you upload an
image through "files", you can right click and select rename to remove the space. Alternatively, you can do this prior to uploading it
to your website.

  1. Select the “Upload Files” icon
  2. Browse to the image, document, or media file you wish to add
  3. Hit “Ok”
  4. Move to desired folder: Documents, Images, Videos
Linking to files
  1. Select the file you have uploaded.
  2. Select the “Get info” tab
  3. Copy the entire URL listed in the “File info” window
  4. Highlight the text you wish to link to (Example: “Download our application”)
  5. Select the connected link icon from the editor
  6. Paste the entire URL into the Link URL field in the Insert/Edit Link field

Or

  1. Edit the page you wish to add a link to a file to
  2. Highlight the text to be a link
  3. Select the hyperlink icon
  4. Select the browse icon to browse your documents folder
  5. Navigate to the documents and select your document
  6. Double click or select the insert icon (arrow)
  7. Hit “Insert”
